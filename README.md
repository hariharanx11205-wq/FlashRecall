# FlashRecall

**FlashRecall** is a minimalist, premium-designed spaced repetition flashcard application. Built with simplicity and performance in mind, it helps you master any subject through efficient study sessions.

## ‚ú® Features
- **Spaced Repetition System (SRS)**: Uses a simplified SM-2 algorithm to schedule reviews at optimal intervals.
- **Glassmorphism Design**: Modern, diverse, and dark-themed UI with smooth animations.
- **Deck Management**: Create, edit, and delete custom flashcard decks.
- **Study Mode**: Interactive 3D card flips with "Again", "Hard", "Good", and "Easy" ratings.
- **Local Persistence**: All data is saved automatically to your browser's LocalStorage.
